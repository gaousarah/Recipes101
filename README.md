# 🍴 Mes Recettes — Guide de déploiement

## Étapes pour mettre l'app sur ton téléphone (100% gratuit)

---

### Étape 1 — Créer un compte GitHub (gratuit)
1. Va sur **github.com**
2. Clique **Sign up** et crée un compte gratuit

---

### Étape 2 — Uploader le projet sur GitHub
1. Une fois connectée, clique sur **"New repository"** (bouton vert)
2. Nomme-le `mes-recettes`, laisse tout par défaut, clique **Create repository**
3. Sur la page suivante, clique **"uploading an existing file"**
4. Glisse-dépose **tous les fichiers** de ce dossier zip
5. Clique **Commit changes**

---

### Étape 3 — Déployer sur Vercel (gratuit)
1. Va sur **vercel.com**
2. Clique **Sign up** → choisis **Continue with GitHub**
3. Clique **"Add New Project"**
4. Sélectionne ton repo `mes-recettes`
5. Vercel détecte automatiquement que c'est Vite/React ✅
6. Clique **Deploy** — attends ~1 minute
7. Tu reçois une URL du type `mes-recettes.vercel.app` 🎉

---

### Étape 4 — Installer sur ton téléphone

**iPhone (Safari) :**
1. Ouvre l'URL dans **Safari** (pas Chrome)
2. Appuie sur le bouton **Partager** ↑
3. **"Sur l'écran d'accueil"**
4. **Ajouter** ✅

**Android (Chrome) :**
1. Ouvre l'URL dans **Chrome**
2. Les **trois points** ⋮ → **"Ajouter à l'écran d'accueil"**
3. **Ajouter** ✅

---

L'app s'ouvre maintenant comme une vraie application, sans barre de navigateur ! 🌙
